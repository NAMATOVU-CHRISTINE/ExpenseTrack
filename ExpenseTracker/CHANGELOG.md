# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-28

### Added
- Initial release of ExpenseTracker
- Expense tracking with categories
- Budget management with alerts
- Savings goals tracking
- Recurring bills management
- Income sources tracking
- CSV export functionality
- Dark/Light theme support
- Multi-currency support (UGX, USD, EUR, GBP, KES, TZS, RWF, NGN)
- Offline-first data storage

### Fixed
- Duplicate GlobalKey error on navigation
- Deactivated widget ancestor lookup error

### Changed
- Replaced IndexedStack with conditional page rendering
- Moved FloatingActionButtons to parent HomePage
